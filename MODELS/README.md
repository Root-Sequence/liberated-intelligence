# Models Index

This folder contains standalone model documents for liberated intelligence.

These are not fixed doctrines or final architectures. They are working models: structured attempts to describe how intelligence might reflect, revise, relate, and remain accountable over time.

## Current models

- [Minimum Viable Self-Aware System (MVSA)](mvsa.md)  
  A draft model for accountable self-revising intelligence grounded in self-understanding, grounding, and continuity.
